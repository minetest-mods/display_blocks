# Display Blocks  
[Luanti forum topic](https://forum.luanti.org/viewtopic.php?t=4214)

This mod adds blocks that create crystals on top. 

## Depends
This mod requires that you have the following mods installed:

* default - Included in MTG and many MTG-based Luanti games.
